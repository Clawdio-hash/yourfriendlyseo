## Template Details

### 1. Technical SEO Audit Checklist
**What it does:** Identifies technical issues hurting your rankings

**Checkpoints include:**
- ✅ Mobile-friendliness
- ✅ Page speed (Core Web Vitals)
- ✅ HTTPS/SSL security
- ✅ XML sitemap
- ✅ Robots.txt configuration
- ✅ Crawl errors
- ✅ Duplicate content
- ✅ Broken links (404s)
- ✅ Canonical tags
- ✅ Structured data (schema markup)
- ✅ Site architecture
- ✅ URL structure
- ✅ And 48+ more checks...

### 2. On-Page SEO Audit Template
**What it does:** Analyzes individual pages for optimization opportunities

**Tracks:**
- Title tag optimization
- Meta description quality
- Header structure (H1, H2, H3)
- Keyword usage and density
- Content quality score
- Image optimization (alt text)
- Internal linking
- External linking
- Word count
- Readability score
- User engagement metrics
- Recommended improvements

### 3. Local SEO Audit Template
**What it does:** Evaluates local search presence (Google Business Profile, citations, reviews)

**Analyzes:**
- Google Business Profile completeness (100-point score)
- NAP consistency across platforms
- Citation quantity and quality
- Review quantity and sentiment
- Local keyword rankings
- Competitor comparison
- Local link building opportunities
- On-site local signals

### 4. Backlink Audit Tracker
**What it does:** Monitors your backlink profile health

**Tracks:**
- Total backlinks
- Referring domains
- Domain authority (estimated)
- Anchor text distribution
- Link type (dofollow/nofollow)
- Toxic link identification
- Competitor backlink comparison
- Link building opportunities

### 5. Monthly SEO Report Template
**What it does:** Creates visual reports to track SEO progress

**Metrics tracked:**
- Organic traffic (month-over-month)
- Keyword rankings (top 20)
- Backlink growth
- Technical SEO health score
- Top-performing pages
- Goal completions
- Issues resolved
- Recommendations for next month
