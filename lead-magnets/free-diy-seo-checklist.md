# Free DIY SEO Checklist for Small Business Owners

**Download & print this checklist to optimize your website step-by-step.**

---

## ✅ SETUP & FOUNDATION (Week 1)

### Google Tools Setup
- [ ] Create/claim Google Search Console account
- [ ] Verify website ownership in Search Console
- [ ] Set up Google Analytics (GA4)
- [ ] Link Google Analytics to Search Console
- [ ] Create/claim Google Business Profile (for local businesses)
- [ ] Verify Google Business Profile

### Technical SEO Audit
- [ ] Test mobile-friendliness: [Google Mobile Test](https://search.google.com/test/mobile-friendly)
- [ ] Check page speed: [PageSpeed Insights](https://pagespeed.web.dev/) (aim for 80+)
- [ ] Verify HTTPS is active (URL starts with https://)
- [ ] Create/verify XML sitemap exists (yoursite.com/sitemap.xml)
- [ ] Submit sitemap to Google Search Console
- [ ] Create robots.txt file (yoursite.com/robots.txt)
- [ ] Check for crawl errors in Search Console

---

## 🔍 KEYWORD RESEARCH (Week 2)

### Find Keywords
- [ ] Brainstorm 5-10 seed keywords (main topics for your business)
- [ ] Use Google autocomplete for long-tail keyword ideas
- [ ] Check "People also ask" boxes for related questions
- [ ] Use AnswerThePublic for question-based keywords
- [ ] Review Google Search Console "Performance" report (existing rankings)
- [ ] Identify 10-20 target keywords (mix of short-tail and long-tail)

### Analyze Competition
- [ ] Google your target keywords
- [ ] Analyze top 10 results (what type of content ranks?)
- [ ] Identify content gaps (what's missing from competitors?)
- [ ] Note search intent (informational, commercial, transactional)

---

## 📝 ON-PAGE SEO (Week 3)

### Homepage Optimization
- [ ] Title tag includes primary keyword (50-60 characters)
- [ ] Meta description is compelling (150-160 characters, includes keyword)
- [ ] URL is clean and descriptive
- [ ] H1 tag includes primary keyword
- [ ] Main content includes keyword naturally (2-5 times)
- [ ] Keyword appears in first 100 words
- [ ] Images have descriptive alt text
- [ ] Internal links to other important pages

### Service/Product Pages (Repeat for Each)
- [ ] Unique title tag with target keyword
- [ ] Unique meta description
- [ ] Clean, keyword-rich URL
- [ ] H1 includes keyword
- [ ] Content is 500+ words (for important pages)
- [ ] Images optimized with alt text
- [ ] Internal links to related pages
- [ ] Call-to-action (CTA) is clear

### Blog Posts
- [ ] Title tag includes target keyword
- [ ] Meta description is compelling
- [ ] URL includes keyword
- [ ] H1 matches title (includes keyword)
- [ ] Use H2/H3 subheadings throughout
- [ ] Content is comprehensive (1,500+ words for competitive topics)
- [ ] Include images/videos to break up text
- [ ] 2-5 internal links to related content
- [ ] 1-3 external links to authoritative sources

---

## 🏠 LOCAL SEO (Week 3 - For Local Businesses)

### Google Business Profile
- [ ] Profile is 100% complete (all fields filled)
- [ ] Business name, address, phone (NAP) are accurate
- [ ] Primary category is most specific option
- [ ] All relevant categories selected
- [ ] Business description includes keywords naturally
- [ ] Business hours are accurate
- [ ] 10+ high-quality photos uploaded (exterior, interior, team, products)
- [ ] Services list is complete
- [ ] Attributes selected (wheelchair accessible, etc.)
- [ ] Respond to all reviews (positive and negative)

### Local Citations
- [ ] NAP is consistent across all platforms
- [ ] Listed on Yelp
- [ ] Listed on Bing Places
- [ ] Listed on Apple Maps
- [ ] Listed on Yellow Pages
- [ ] Listed on Better Business Bureau (if applicable)
- [ ] Listed in industry-specific directories

### Reviews
- [ ] Ask 5-10 customers for Google reviews
- [ ] Set up automated review request emails
- [ ] Respond to all reviews within 24-48 hours
- [ ] Display reviews on website (optional)

---

## 📊 TECHNICAL SEO (Week 4)

### Site Speed Optimization
- [ ] Compress images (use TinyPNG or similar)
- [ ] Enable browser caching
- [ ] Minimize CSS/JavaScript files
- [ ] Use a Content Delivery Network (CDN) if needed
- [ ] Optimize web hosting (upgrade if loading times are slow)

### Mobile Optimization
- [ ] Site is responsive on all devices
- [ ] Text is readable without zooming
- [ ] Buttons/links are easy to tap
- [ ] No horizontal scrolling
- [ ] Pop-ups don't block content on mobile

### Site Structure
- [ ] Navigation is clear and intuitive
- [ ] Important pages are 3 clicks or fewer from homepage
- [ ] Broken links are fixed (404 errors)
- [ ] Redirects are in place for deleted pages (301 redirects)
- [ ] Internal linking structure connects related content

---

## ✍️ CONTENT STRATEGY (Month 2+)

### Content Planning
- [ ] Create content calendar (1-2 posts per week)
- [ ] Plan 3-5 pillar posts (comprehensive, 2,000+ words)
- [ ] Plan 10+ supporting posts (cluster content)
- [ ] Match content to search intent (informational, commercial, transactional)

### Content Creation
- [ ] Write for humans first, search engines second
- [ [ ] Answer the searcher's question completely
- [ ] Add unique value (don't just copy competitors)
- [ ] Use short paragraphs (2-4 sentences)
- [ ] Include bullet points and numbered lists
- [ ] Add images, videos, or infographics
- [ ] Proofread and edit before publishing

### Content Optimization
- [ ] Target one primary keyword per page
- [ ] Include keyword in title, H1, first 100 words
- [ ] Use related keywords (LSI keywords) naturally
- [ ] Add internal links to related content
- [ ] Include external links to authoritative sources
- [ ] Add clear CTAs (call-to-actions)

---

## 🔗 LINK BUILDING (Month 2+)

### Backlink Strategy
- [ ] Create 1-2 link-worthy resources (guides, tools, infographics)
- [ ] Build citations in 10-20 directories
- [ ] Reach out for 5 guest post opportunities
- [ ] Partner with complementary businesses for links
- [ ] Reclaim unlinked brand mentions (Google your brand)
- [ ] Sponsor local events/charities for links

### Monitor Backlinks
- [ ] Check backlinks monthly in Google Search Console
- [ ] Disavow toxic/spammy backlinks (if necessary)

---

## 📈 TRACKING & REPORTING (Ongoing)

### Weekly Check-In
- [ ] Review Google Analytics traffic (organic sessions)
- [ ] Check Search Console for new ranking keywords
- [ ] Monitor top-performing pages

### Monthly Review
- [ ] Track keyword rankings (top 20 target keywords)
- [ ] Review backlink growth
- [ ] Analyze top traffic pages
- [ ] Identify opportunities (pages on page 2 that can be optimized)
- [ ] Update old content with fresh information

### Quarterly Audit
- [ ] Full technical SEO audit (speed, mobile, errors)
- [ ] Content audit (update or remove outdated posts)
- [ ] Backlink audit (identify and disavow toxic links)
- [ ] Adjust strategy based on what's working

---

## 🎯 90-DAY QUICK START PLAN

### Month 1: Foundation
- **Week 1:** Setup (Google tools, technical audit)
- **Week 2:** Keyword research
- **Week 3:** Optimize homepage + top 3 pages
- **Week 4:** Publish 2 pillar blog posts

### Month 2: Content & Links
- **Week 1-4:** Publish 8 blog posts (2/week)
- **Week 1-4:** Build 10+ citations
- **Week 1-4:** Request reviews
- **Week 1-4:** Guest post outreach (5+ sites)

### Month 3: Scale & Optimize
- **Week 1-4:** Continue 2 posts/week
- **Week 1-4:** Update 2-4 old posts
- **Week 1-4:** Build 5-10 backlinks
- **Week 1-4:** Monitor & adjust strategy

---

## 🚫 AVOID THESE COMMON MISTAKES

- [ ] ❌ Don't keyword stuff (use keywords naturally)
- [ ] ❌ Don't publish thin content (<300 words)
- [ ] ❌ Don't ignore mobile users
- [ ] ❌ Don't copy competitors (add unique value)
- [ ] ❌ Don't buy backlinks (earn them)
- [ ] ❌ Don't expect overnight results (SEO takes 3-6 months)
- [ ] ❌ Don't ignore user experience (fast, clean, easy to navigate)

## 🛠️ RECOMMENDED FREE TOOLS

### SEO Research
- Google Search Console (essential)
- Google Analytics (essential)
- Google Keyword Planner
- AnswerThePublic (3 free searches/day)
- Google Trends

### Technical SEO
- Google Mobile-Friendly Test
- Google PageSpeed Insights
- Screaming Frog SEO Spider (free for 500 URLs)

### Content
- Grammarly (free version)
- Hemingway Editor (readability)
- Canva (free images/graphics)

### Backlinks
- Google Search Console (backlink report)
- Google Alerts (monitor mentions)
