## **Content Production Workflow**

### For Each Blog Post:
1. **Research Phase:**
   - Keyword research (target + LSI keywords)
   - Competitor content analysis (top 5 ranking pages)
   - Identify content gaps

2. **Writing Phase:**
   - Create outline
   - Write draft (hit target word count)
   - Optimize for target keyword (title, meta, headers, first 100 words)
   - Add internal links to related posts
   - Add external links to authoritative sources

3. **SEO Optimization:**
   - Meta title (50-60 chars, include target keyword)
   - Meta description (150-160 chars, include keyword + CTA)
   - URL slug (lowercase, hyphens, target keyword)
   - Image alt text (descriptive + keyword when relevant)
   - Schema markup (Article schema)

4. **Review & Publish:**
   - Commit to `content` branch
   - Wait for human review
   - Merge to `main` after approval
   - Monitor rankings + traffic

### For Digital Products:
1. **Creation Phase:**
   - Design template/course outline
   - Build product files (Google Sheets, PDF, video, etc.)
   - Write sales page copy

2. **Launch Phase:**
   - Set up Stripe payment link
   - Create landing page
   - Write launch blog post
   - Announce via email list + social

3. **Marketing Phase:**
   - Add CTAs to related blog posts
   - Create lead magnet (free version)
   - Email sequence for upsell
